Thank you for purchasing the Pixel UI Kit!

This pack includes 30 handcrafted pixel-art UI elements, perfect for retro and minimalist games.

-----------------------------------
📦 CONTENTS
-----------------------------------

- PNG sprites (32x32px, transparent background)
- Multiple UI states: Normal, Hovered, Pressed, Marked
- Icons: Play, Pause, Settings, Heart, etc.
- Elements: Buttons, Checkboxes, Radio buttons, Dropdowns, Sliders, Dialogues
- BONUS: Window frame, Cursor, Soft-style buttons

-----------------------------------
🧰 HOW TO IMPORT
-----------------------------------

1. Drag the entire folder "PixelUI" into your Unity project's "Assets" folder.
2. Unity will automatically import all PNG files as sprites.
3. Don't forget to set import settings (Sprite/UI) for sprites in Unity!
4. Set filter mode to Point (No Filter) in the import settings

-----------------------------------
🛠 HOW TO SET UP
-----------------------------------

1. Create a new Unity UI Canvas (Right-click > UI > Canvas)
2. Set Canvas to "Screen Space - Overlay"
3. Create a new Image or Button (UI > Image or UI > Button)
4. Replace its Source Image with one of the PNGs from this pack
5. Adjust "Image Type" to:
   - Simple: for square buttons/icons
   - Sliced: for scalable windows/dialogues (like window_frame)

Tip: Add different sprite states to Unity's Button component (Normal, Highlighted, Pressed) manually.
